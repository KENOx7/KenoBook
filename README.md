<h2 id="english">🇬🇧 English</h2>

A modern, high-performance Personal Link & Note Manager with a beautiful dual-theme UI, embedded video support, and interactive particle animations.

### What's New in v1.0.0 🎉
- **🌗 Light / Dark Mode:** Seamlessly switch between a sleek dark theme and a clean, modern light theme.
- **📺 Embedded Videos:** YouTube and Vimeo links automatically render as inline video players.
- **✍️ Rich Text Toolbar:** Format your notes visually with Bold, Italic, Heading, List, Code, and Link buttons — no Markdown knowledge needed.
- **🎨 Interactive Particle Background:** Animated canvas with mouse-reactive physics (desktop only).
- **🔐 PIN Security & Rate Limiting:** Protect your data with a built-in PIN verification system.
- **📐 Masonry Layout:** Pinterest-style card arrangement for a clean, gap-free design.

### Tech Stack 🛠
| Layer | Technology |
|-------|-----------|
| Frontend | React 18 (Vite.js) |
| Styling | Tailwind CSS v4 |
| Backend | Supabase |
| Icons | Lucide React |
| Markdown | React-Markdown & Remark-GFM |

### Quick Start 🚀
```bash
# 1. Clone the repo
git clone https://github.com/YOUR_USERNAME/KenoBook.git
cd KenoBook

# 2. Install dependencies
npm install

# 3. Create .env file (NEVER push this to GitHub!)
# Add these keys:
#   VITE_SUPABASE_URL=your_url
#   VITE_SUPABASE_ANON_KEY=your_key
#   VITE_SECURITY_PIN=your_pin

# 4. Run dev server
npm run dev
```

### Deployment 🌐
For Vercel / Netlify, add `.env` keys to **Settings → Environment Variables** before deploying.
